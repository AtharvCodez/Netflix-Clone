# Netflix-Clone
A responsive Netflix-inspired landing page built using HTML and CSS as my first frontend development project, focusing on layout structure and UI replication.

**Sep 2025**

A responsive Netflix Clone that recreates the original platform’s look and feel with modern styling and smooth animations. The homepage features structured UI sections and a fully designed sign-in page to deliver an authentic user experience.

---

## 🚀 Features

1. **Hero Section** with animated intro text and interactive email input form  
2. **Content Highlight Cards** showcasing key streaming features (TV, Offline Viewing, Kids Profiles, etc.)  
3. **FAQ Section** with smooth hover animations  
4. **Responsive Sign-In Page** with floating labels and scaling effects  
5. **Navigation Bar** with interactive buttons and hover transitions  
6. **Structured Footer** with categorized links similar to Netflix’s official layout
